3.1.1.4. ThinkpadAcpiBatteryControl¶

This module provides an interface to ACPI Calls to control Thinkpad Laptop Battery.

These ACPI calls permits: * to set the start and stop capacity threshold to charge the battery, * to switch on battery when AC power is plugged, * to setup a “peak shift” procedure.

(As far I know) The concept of “peak shift” is to switch temporarily electrical devices on battery during a power peak consumption period so as to unload the grid. This power management strategy is relevant for country like Japan, where these peak periods represent a risk of electrical black out.
